# Branding (logo công ty)

Repo đã có **`logo.svg`** mặc định (chữ TH).

1. Thay logo công ty: ghi đè `logo.svg` **hoặc** đặt `logo.png` / `.jpg` / `.webp`.
2. Nếu đổi tên file → sửa `logo_file` trong `config.json` (vd. `web/branding/logo.png`).
3. Gợi ý: PNG trong suốt, cao ~120–200 px.
4. Khởi động lại app — logo hiện sidebar + chứng từ.

Không bắt buộc: nếu không có file logo, UI dùng chữ **TH**.
