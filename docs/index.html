Phân quyền chặn ở API (không chỉ ẩn nút). Seed lần đầu: mật khẩu ngẫu nhiên in ra console → bắt đổi khi login.

Ảnh trên là mockup tương tác trong trình duyệt (HTML/CSS). App thật sau khi setup.bat chạy tại http://127.0.0.1:8777.

Module nghiệp vụ

Một bản cài phục vụ cả văn phòng kỹ thuật, kế toán và giám đốc.

Báo giá & BOQ

Revision, VAT footer, chuẩn hóa hạng mục, xuất chứng từ.

Công trình & nhật ký

Pipeline CT, KTV ghi nhật ký, KTT theo dõi tiến độ.

Công nợ & TT

Phải thu/trả, xác nhận sao kê, TT ngoài công ty có vết.

Hồ sơ CT

Checklist, export bộ hồ sơ, quét folder khách trên ổ đĩa.

Kho & vật tư

Xuất nhập, tồn, giá vật tư — thủ kho / kỹ thuật.

RBAC & audit

7+ vai trò, đổi mật khẩu bắt buộc, force-logout.

Cài & chạy (máy Windows)

  1. Cài Python 3.11+ (tick Add to PATH)
  2. Tải / clone repo GitHub
  3. Double-click setup.bat → ghi mật khẩu in ra màn hình
  4. Mở http://127.0.0.1:8777 → đổi mật khẩu → sửa config.json (tên + logo)
git clone https://github.com/Jokejoker-designer/ERPcompany-.git

Sau khi cài

  • Logo: web/branding/logo.svg (hoặc PNG)
  • Tên công ty: config.json
  • Quét hồ sơ: scan_roots
  • Lần sau: chỉ cần start.bat
