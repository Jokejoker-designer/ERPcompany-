Hệ quản trị nội bộ chạy trên máy bạn

Thanh Hoai ERP là phần mềm quản trị doanh nghiệp cài trực tiếp trên Windows: dữ liệu SQLite nằm tại máy / server nội bộ, mặc định chỉ lắng nghe localhost. Tạo nhiều tài khoản (Giám đốc, Kế toán, Kinh doanh, KTT, KTV, Thủ kho…) với phân quyền kiểm soát ở API — không chỉ ẩn nút trên giao diện.

Local-first · Nhiều người dùng · Kiểm soát truy cập

Những điểm dưới đây có thật trong mã nguồn package — không quảng cáo tính năng chưa có.

Chạy local, dữ liệu ở bạn

Ứng dụng là web server nhỏ trên Windows. Mặc định bind 127.0.0.1 (chỉ máy cài). CSDL một file SQLite trong thư mục data/ — không đẩy lên cloud của nhà cung cấp.

Nhiều tài khoản, phân quyền thật

Seed sẵn các vai trò (admin, giám đốc, kế toán, kinh doanh, KTT, KTV, thủ kho). Có thể cấp thêm tài khoản trong hệ thống. Quyền xem/sửa được kiểm tra phía API; mật khẩu dùng scrypt và bắt đổi khi đăng nhập lần đầu.

Dùng chung trong đội — khi bạn tự mở cổng an toàn

Package không kèm Cloudflare hay Tailscale sẵn. Vì app chỉ là HTTP trên một cổng, IT / admin có thể tự gắn Cloudflare Tunnel hoặc Tailscale Serve trỏ về máy cài (HTTPS, kiểm soát ai vào mạng) — giống cách expose mọi dịch vụ local. Cấu hình đó do bạn / đơn vị triển khai, không phải nút “bật cloud” trong ERP.

Giao diện vận hành — xem trước

Khám phá các màn hình chính: dashboard điều hành, công nợ, báo giá, bộ hồ sơ công trình và phân quyền. Chọn tab hoặc menu bên trái để chuyển màn.

Triển khai trên Windows

Chỉ vài phút để có hệ thống chạy tại chỗ — dữ liệu thuộc về doanh nghiệp bạn.

  1. Cài Python 3.11+ (tick Add to PATH)
  2. Tải mã nguồn từ GitHub (clone hoặc ZIP)
  3. Chạy setup.bat — ghi lại mật khẩu khởi tạo
  4. Cập nhật config.json — tên công ty, logo, thư mục hồ sơ
